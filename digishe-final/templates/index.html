{% if current_lang == 'en' %} About DigiShe {% else %} DigiShe ಬಗ್ಗೆ {% endif %}

{% if current_lang == 'en' %} DigiShe is a digital platform dedicated to making government schemes for women more accessible. We believe that information accessibility is the first step towards empowerment. {% else %} DigiShe ಮಹಿಳೆಯರಿಗೆ ಸರ್ಕಾರಿ ಯೋಜನೆಗಳನ್ನು ಹೆಚ್ಚು ಲಭ್ಯವಾಗುವಂತೆ ಮಾಡಲು ಸಮರ್ಪಿತವಾದ ಡಿಜಿಟಲ್ ವೇದಿಕೆಯಾಗಿದೆ. ಮಾಹಿತಿ ಪ್ರವೇಶಾವಕಾಶವು ಸಬಲೀಕರಣದ ಮೊದಲ ಹೆಜ್ಜೆ ಎಂದು ನಾವು ನಂಬುತ್ತೇವೆ. {% endif %}
